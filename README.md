# 旅行プラン提案アプリ

あなたの希望に合わせた旅行プランを3案提案するWebアプリケーションです。**APIキー不要で無料で利用できます！**

## 機能

- 入力フォーム：希望要素（温泉/秘境/歴史建造物/グルメ/自然/アート）、出発地、行きたい方面（任意）、予算、日程、時期
- 日本全国の観光地データベースから旅行先を3案提案
- 予算に応じた適切なプラン生成（リーズナブル/スタンダード/プレミアム）
- 出発地と行きたい方面を考慮した目的地選択
- 季節に応じたおすすめ情報の表示
- 提案結果をローカルストレージに保存・閲覧可能
- ワクワクする旅行っぽいデザイン（爽やか・明るい色使い）
- レスポンシブ対応

## 使い方

1. アプリを開く（APIキー不要！）
2. 希望要素、出発地、行きたい方面（任意）、予算、日程、時期を入力
3. 「旅行プランを提案してもらう」ボタンをクリック
4. 提案された3つのプランから選ぶ

## GitHub Pagesでのデプロイ方法

### 1. GitHubリポジトリを作成

```bash
# リポジトリを初期化
git init
git add .
git commit -m "Initial commit"

# GitHubでリポジトリを作成後、以下を実行
git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
git branch -M main
git push -u origin main
```

### 2. GitHub Pagesを有効化

1. GitHubリポジトリのページにアクセス
2. 「Settings」タブをクリック
3. 左メニューから「Pages」を選択
4. 「Source」で「Deploy from a branch」を選択
5. 「Branch」で「main」を選択し、「/ (root)」を選択
6. 「Save」をクリック

### 3. アクセス

数分後、以下のURLでアクセスできます：
```
https://あなたのユーザー名.github.io/リポジトリ名/
```

## ローカルでの実行方法

### Pythonを使用（推奨）

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス

### Node.jsを使用

```bash
npx serve
```

## 特徴

- **APIキー不要**: 完全無料で利用可能
- **日本全国対応**: 津々浦々の観光地から提案
- **予算に応じた最適化**: 1日の予算に応じて適切なプランを生成
- **出発地を考慮**: 出発地に近い目的地を優先的に提案
- **行きたい方面の指定**: 希望の地域を指定すると、その地域の観光地を優先
- **季節に応じた情報**: 春・夏・秋・冬それぞれに適したおすすめ情報を表示
